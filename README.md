🌐 Port Scanner

📌 Project Overview
The Port Scanner is a Python-based tool that scans a target system to identify open ports and running services.  
It helps in understanding network exposure and is useful for basic security analysis and penetration testing.

🚀 Features
- 🔍 Scan open ports on a target system
- ⚡ Fast scanning using multi-threading (if implemented)
- 🌐 Supports scanning of common ports (e.g., 21, 22, 80, 443)
- 📊 Displays open/closed port status
- 🛠️ Lightweight and easy to use

# 🛠️ Technologies Used
- Python
- Socket Programming
- Threading (optional)
